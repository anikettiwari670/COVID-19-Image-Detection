# 🫁 COVID-19 X-Ray Image Detection

A deep learning project to classify chest X-ray images as Covid, Normal, or Viral Pneumonia using a CNN.

## Models Trained
| Model | Test Accuracy | F1 Score |
|---|---|---|
| Basic CNN ✅ | 92.06% | 0.92 |
| VGG16 | 33.33% | 0.17 |
| ResNet50 + Augmentation | 31.75% | 0.15 |

## Tech Stack
Python · TensorFlow · Keras · OpenCV · Streamlit · Google Colab

## Dataset
Kaggle — Covid19 Image Dataset (251 chest X-ray images)
